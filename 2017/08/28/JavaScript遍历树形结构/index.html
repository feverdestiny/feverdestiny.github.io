在 JavaScript 中也很多树形结构。比如 DOM 树，省市区地址联动，文件目录等； JSON 本身就是树形结构。
很多前端面试题也跟树形结构的有关，比如在浏览器端写遍历 DOM 树的函数，比如在 nodejs 运行时遍历文件目录等。
